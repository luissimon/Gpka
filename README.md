# Gpka

scripts and data for the file in: https://doi.org/10.26434/chemrxiv-2025-sllx4/v2

Data is also available in: https://orwell.usal.es/Gpka including files that are too large to be included in github:

 -output_files:   the (thousands) of output files from QM calculations
 
 -models:         the models described in the preprint
 
 -extracted data: the data obtained after runing some of the scripts on the output files
